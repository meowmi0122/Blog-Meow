# GitHub 使用教學＆功能介紹

## 什麼是 GitHub？

GitHub 是一個可以存放程式碼、網站、專案的平台。

你可以用它：

- 建立網站
- 備份檔案
- 分享作品
- 管理版本
- 與他人開發
- 放置圖片與程式碼

---

# GitHub 功能介紹

## Repository（儲存庫）

Repository 是專案資料夾。

裡面可以放：

- HTML
- CSS
- JavaScript
- Python
- Markdown
- 任何格式程式碼
- 圖片
- App 專案

---

## Commit（版本紀錄）

每次修改檔案後都能建立紀錄。

例如：

Update Website  
Fix Bug  
Add New Page

---

## Branch（分支）

可以建立不同版本。

例如：

main  
dev  
beta

---

## Pull Request

可以把分支內容合併回主分支。

用途：

- 團隊協作
- 檢查程式碼
- 測試功能

---

## GitHub Pages

可以把網站變成公開網址。

例如：

https://username.github.io

---

## README.md

README 是專案首頁介紹檔。

支援：

- 標題
- 圖片
- SVG
- 表格
- 程式碼區塊
- 超連結

---
